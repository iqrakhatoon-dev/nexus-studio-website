# NEXUS.Studio ✦ Design Agency Website

A premium design agency website inspired by 
[Lazarev.agency](https://lazarev.agency) — built 
with HTML, CSS, JavaScript and GSAP animations.

## 🔗 Live Demo
[vercel-link](https://nexus-studio-website.vercel.app/)

## ✨ Features
- Smooth locomotive scroll
- GSAP scroll-triggered animations
- Animated loading screen
- Navbar mega dropdown with slide animation
- Marquee ticker bar
- Blog section with hover image effect
- Showreel video reveal
- Case studies with video hover effect
- Services accordion (details/summary)
- Design process steps section
- Fully responsive (mobile + tablet)

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- GSAP 3 + ScrollTrigger
- Locomotive Scroll
- Font Awesome Icons
- Google Fonts (Syne)
